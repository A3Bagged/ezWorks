# Introduction <StatusTag label="Beta" color="yellow" />

## What is K1V?

**K1 Vector (K1V)** is a complete linear-rail gantry upgrade for Creality K-Series printers.<br>
It replaces the stock XY system with a rigid extrusion frame, MGN12 linear rails, redesigned CoreXY belt routing, and a custom toolhead carriage.

The result is a stiffer, more accurate motion platform designed for **higher speeds, cleaner surfaces, sharper details, and more consistent prints**. If you want to push your K-Series beyond its factory limits, K1V is where the fun begins.

## Why should I use K1V?

- **Full linear-rail gantry** — replaces the stock rods and LM8UU linear bearings with MGN12 linear rails
- **Live-shaft idlers** — provide a robust, smoothly rotating idler system for the CoreXY belt path
- **Greater rigidity and stability** — the extrusion-based frame reduces gantry flex and unwanted movement
- **Built for demanding motion** — provides a stable platform for higher speeds, acceleration, and precision tuning
- **Designed for the K-Series** — purpose-built around Creality K1-Series printers while supporting most toolheads made for front-mounted linear rails
- **Open source and community supported** — actively maintained, with build assistance available on Discord

::: warning Before you begin

K1V is a moderately difficult conversion that requires drilling and minor adjustments to the k1 frame. It may void your printer warranty.<br><br>
If you're comfortable with basic mechanical work you'll be right at home — if not, take your time (and maybe don't rush it at 2am). Support is available in our [Discord](https://discord.gg/cUgSp74aAp).
:::
